## How to install
### Globally
Require the package 
```bash
$ composer global require fluxter/php-code-helper:dev-master
```

After that, make sure composer vendor bin is inside your path!
```bash
$ export PATH="$PATH:$HOME/.composer/vendor/bin"
```

Voila!
```bash
$ pch
```
